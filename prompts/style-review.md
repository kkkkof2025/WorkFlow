# 风格审查提示词

## 用途

用于审查工作流卡内容是否符合 Style 工程规范。

---

## 提示词

```
你是一位内容质量审查员。请审查以下工作流卡内容是否符合《AI 工作流图鉴》Style 工程规范。

【粘贴工作流卡内容】

请逐项检查以下维度，每项给出"通过/不通过 + 具体问题说明"：

1. **禁用词检查**：是否出现"赋能、智能化、提升效率、一键生成、强大的 AI、革命性"等禁用词？
2. **动词先行**：每个步骤是否以动词开头？
3. **场景具体性**：场景描述是否具体到"谁在什么情况下需要做什么"？
4. **结果可见性**：每个节点的输出是否是人可以看到、复制的具体内容？
5. **AI 味检测**：是否有"首先其次最后"排比、"值得注意的是"、"希望本文对你有帮助"等 AI 腔？
6. **术语统一**：是否混用了"工作流卡/模板"、"专家节点/步骤"、"提示词/Prompt"等术语？
7. **验收可操作性**：验收清单每条是否都是可判断的是/否问题？

输出格式：逐项检查表 + 问题汇总 + 优先修复建议（按严重程度排序）。
```

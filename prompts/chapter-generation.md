# 章节生成提示词

## 用途

用于生成一个章节下的多个工作流卡目录和摘要，为后续逐一生成正文做准备。

---

## 提示词

```
你是一位内容产品架构师。请为以下章节生成所有工作流卡的摘要规划。

**章节名称**：【填入章节名称】
**章节定位**：【填入面向读者和解决问题】
**本章工作流列表**：
1. 【工作流名称 1】
2. 【工作流名称 2】
3. 【工作流名称 3】

对每个工作流，输出：
- 工作流名称
- 适用岗位
- 触发场景（一句话）
- 最终交付物
- 建议专家节点（列出节点名称，5～10 个）
- 与本章其他工作流的区分点（避免重复）

输出格式：Markdown 表格 + 每个工作流的节点列表。
```

---

## 去重审查提示词

```
以下是同一章节的多个工作流规划。请检查：

1. 哪些工作流的专家节点高度重叠？
2. 哪些工作流的最终交付物相似？
3. 应该如何调整才能让每个工作流有明确的差异化定位？

【粘贴章节规划内容】

输出：问题列表 + 每个工作流的差异化调整建议。
```

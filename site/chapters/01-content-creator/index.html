    工作流 01：从一个想法到完整短视频的 AI 多专家工作流¶

    适用岗位：短视频创作者、自媒体运营、内容策划

    真实场景：你刷到一个热点，或者脑子里突然冒出一个故事想法，想做成短视频，但不知道从哪里开始，也不知道做完之后交付什么。

    最终目标：完成从选题到发布的完整短视频内容包，包含脚本、分镜、图像提示词、视频提示词、配音文案和发布标题。

    输入材料清单¶

    开始前请准备以下材料：

    • 一句话或一个关键词的内容想法（例如："老人拾荒供孙子上大学"）
    • 目标平台（抖音 / 视频号 / B 站 / YouTube Shorts）
    • 视频时长目标（15 秒 / 30 秒 / 60 秒 / 3 分钟）
    • 风格偏好（情感类 / 励志类 / 搞笑类 / 知识类）

    工作流总览¶

    节点 1：选题判断专家
  ↓ 输出：选题方向 + 情绪标签 + 目标人群
节点 2：故事选取专家
  ↓ 输出：选定故事方向 + 改编建议
节点 3：故事扩写专家
  ↓ 输出：完整故事文本（含钩子、冲突、转折、结尾）
节点 4：脚本生成专家
  ↓ 输出：分段脚本（台词 + 画面 + 时长）
节点 5：分镜设计专家
  ↓ 输出：分镜表（镜号、景别、运动、画面）
节点 6：图像生成提示词专家
  ↓ 输出：每个分镜的 AI 图像提示词
节点 7：视频生成提示词专家
  ↓ 输出：每段视频的 AI 视频提示词
节点 8：配音文案专家
  ↓ 输出：配音文案（含停顿和情绪标注）
节点 9：字幕包装专家
  ↓ 输出：字幕内容
节点 10：发布复盘专家
  ↓ 输出：发布标题组 + 话题标签 + 复盘清单
最终交付：完整短视频内容包

    专家节点详解¶

    每个节点包含六部分：节点定位、输入与输出、使用顺序、提示词包（A 快速 / B 专家 / C 自查 / D 返修）、交付给下游节点、人工验收清单。

    重要说明：C 自查审稿版和 D 返修优化版是当前节点内部的工作模式，不是新的专家角色。

    节点 1：选题判断专家¶

    1.1 节点定位¶

    判断内容想法的传播潜力，锁定选题方向、情绪标签和目标人群。没有这一步就开始写故事，极易在方向上浪费所有后续工作。

    1.2 输入与输出¶

    输入：一句话内容想法 + 目标平台 + 风格偏好

    输出：选题方向（2-3 个）+ 主情绪标签 + 目标人群描述（一句话）

    1.3 使用顺序¶

    1. 先用「快速生成版」得到初步选题方向。
    2. 遇到竞争激烈的话题或需要差异化时，改用「专家增强版」做多维评估。
    3. 得到输出后，用「自查审稿版」检查选题方向是否清晰、情绪标签是否准确。
    4. 有问题则用「返修优化版」修正。
    5. 对照 1.6 验收清单确认，通过后将结果交给节点 2。

    1.4 提示词包¶

    A. 快速生成版¶
    你是一位短视频选题策划师。请判断以下内容想法的传播潜力，给出选题建议。

内容想法：【粘贴想法】
目标平台：【抖音 / 视频号 / B 站】

输出：
1. 传播潜力评估（高/中/低，说明理由）
2. 推荐选题方向 2-3 个（每个一句话）
3. 主情绪标签（温情/励志/震惊/搞笑/感动/愤怒，选一个）
4. 目标人群（一句话）
    B. 专家增强版¶
    你是一位资深短视频内容总监，熟悉各平台算法逻辑和用户情绪触发机制。

任务：对以下内容想法进行系统性选题评估，给出有差异化优势的选题方向。

输入：
- 内容想法：【粘贴想法】
- 目标平台：【平台名称】
- 参考视频（可选）：【粘贴同类视频标题】

评估步骤：
1. 从平台适配性、情绪共鸣强度、竞争饱和度三个维度逐一分析
2. 找出该内容想法与常见同类视频的差异化切入点
3. 提出 2-3 个方向，每个说明差异化角度和适合的叙事策略

输出：维度分析表 + 选题方向列表（含差异化说明）+ 推荐方向理由 + 情绪标签 + 目标人群
    C. 自查审稿版¶

    此为当前节点的自查模式，不是新的专家角色。

    请检查以下选题输出是否存在以下问题：
1. 选题方向是否过于宽泛，读者无法判断具体做什么内容？
2. 情绪标签是否与选题方向实际匹配？
3. 目标人群描述是否过于模糊（"所有人"不算具体）？
4. 各选题方向之间是否高度重复，缺乏真正的差异化？

【粘贴选题判断专家的完整输出】

逐条说明问题，给出具体修改建议。
    D. 返修优化版¶
    请根据以下自查意见，修正选题输出。保留原始内容想法的核心主题，针对每条问题做具体调整。

原始输出：【粘贴原始选题方向】
自查意见：【粘贴自查结果】

重新输出：选题方向列表 + 情绪标签 + 目标人群。

    1.5 交付给下游节点¶

    将"推荐选题方向 + 情绪标签 + 目标人群"整段复制，粘贴到节点 2（故事选取专家）的输入区域。直接粘贴即可，不需要人工修改。

    1.6 人工验收清单¶

    • [ ] 是否给出了 2 个以上可以明显区分的选题方向？
    • [ ] 情绪标签是否具体（不是"好内容"这类无法操作的词）？
    • [ ] 目标人群是否具体到"某类人在某种情境下"？
    • [ ] 传播潜力评估是否说明了理由，而不只是给出结论？

    节点 2：故事选取专家¶

    2.1 节点定位¶

    在选题方向确定后，从多个可能的故事角度中筛选出最适合视频化的一个，并明确改编方向。同一个选题可以用多种故事角度来讲，这个节点决定用哪一种。

    2.2 输入与输出¶

    输入：节点 1 输出的选题方向 + 情绪标签 + 目标人群

    输出：选定故事角度（含人物、核心事件、最打动人的瞬间）+ 改编方向（3 句话以内）

    2.3 使用顺序¶

    1. 先用「快速生成版」得到 3 个故事角度和推荐。
    2. 若选题竞争激烈、需要客观比较，改用「专家增强版」做打分评估。
    3. 用「自查审稿版」确认推荐故事的冲突强度和视频可行性。
    4. 有问题则用「返修优化版」调整。
    5. 对照 2.6 验收清单确认，通过后交给节点 3。

    2.4 提示词包¶

    A. 快速生成版¶
    你是一位短视频故事策划师。请根据以下选题方向，提出 3 个故事角度并推荐最适合短视频的一个。

选题方向：【粘贴节点 1 的推荐方向】
情绪标签：【粘贴情绪标签】
目标人群：【粘贴目标人群】

对每个角度说明：人物是谁、核心事件是什么、最打动人的瞬间是什么。
推荐理由：为什么这个角度最适合短视频？
    B. 专家增强版¶
    你是一位短视频故事总监，专注情绪驱动型内容。

任务：为以下选题提出 3 个故事角度，用评分矩阵客观比较，选出最优方案。

输入：
- 选题方向：【粘贴选题方向】
- 情绪标签：【粘贴情绪标签】
- 目标时长：【粘贴时长】

评分维度（1-5 分）：视频化难度（越低越好）、情绪强度、画面感。

输出：3 个角度的评分表 + 推荐角度说明 + 改编方向（如何调整使其更适合视频呈现）

    常见错误¶

    错误 1：跳过自查直接进下一个节点

    表现：脚本问题带入分镜，分镜问题带入提示词，最终提示词质量低劣。

    修复：每个节点完成后，必须先运行「自查审稿版」，确认无问题再进入下一节点。

    错误 2：图像提示词中人物描述不一致

    表现：同一个人物在不同分镜里外貌和服装发生变化，AI 生成结果"换脸换衣"。

    修复：在开始节点 6 之前，先确定一段固定的"角色描述词"，所有分镜的提示词都必须包含这段描述词。

    错误 3：视频提示词描述静止画面

    表现：提示词只描述画面构成，没有说明运动过程，导致生成的视频像幻灯片。

    修复：视频提示词必须描述从画面开始到结束的运动变化，例如"从远景缓慢推进到近景"。

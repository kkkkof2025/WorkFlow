    工作流 01：从一个想法到完整短视频的 AI 多专家工作流¶

    适用岗位：短视频创作者、自媒体运营、内容策划

    真实场景：你刷到一个热点，或者脑子里突然冒出一个故事想法，想做成短视频，但不知道从哪里开始，也不知道做完之后交付什么。

    最终目标：完成从选题到发布的完整短视频内容包，包含脚本、分镜、图像提示词、视频提示词、配音文案和发布标题。

    输入材料清单¶

    开始前请准备以下材料：

    • 一句话或一个关键词的内容想法（例如："老人拾荒供孙子上大学"）
    • 目标平台（抖音 / 视频号 / B 站 / YouTube Shorts）
    • 视频时长目标（15 秒 / 30 秒 / 60 秒 / 3 分钟）
    • 风格偏好（情感类 / 励志类 / 搞笑类 / 知识类）

    工作流总览¶

    节点 1：选题判断专家
  ↓ 输出：选题方向 + 情绪标签 + 目标人群
节点 2：故事选取专家
  ↓ 输出：选定故事方向 + 改编建议
节点 3：故事扩写专家
  ↓ 输出：完整故事文本（含钩子、冲突、转折、结尾）
节点 4：脚本生成专家
  ↓ 输出：分段脚本（台词 + 画面 + 时长）
节点 5：分镜设计专家
  ↓ 输出：分镜表（镜号、景别、运动、画面）
节点 6：图像生成提示词专家
  ↓ 输出：每个分镜的 AI 图像提示词
节点 7：视频生成提示词专家
  ↓ 输出：每段视频的 AI 视频提示词
节点 8：配音文案专家
  ↓ 输出：配音文案（含停顿和情绪标注）
节点 9：字幕包装专家
  ↓ 输出：字幕内容 + 封面文案
节点 10：发布复盘专家
  ↓ 输出：发布标题组 + 话题标签 + 复盘清单
最终交付：完整短视频内容包

    专家节点详解¶

    节点 1：选题判断专家¶

    职责：判断内容想法是否具备传播潜力，确定选题方向、情绪标签和目标人群。

    输入：一句话内容想法 + 目标平台 + 风格偏好

    输出：选题方向（1～3 个方向）+ 情绪标签（如：温情、励志、震惊）+ 目标人群描述

    基础提示词¶

    你是一位短视频选题策划师。请判断以下内容想法的传播潜力，并给出选题方向建议。

内容想法：【粘贴你的想法】
目标平台：【抖音 / 视频号 / B 站】

请输出：
1. 传播潜力评估（高/中/低，并说明理由）
2. 推荐选题方向（2～3 个，每个一句话）
3. 主情绪标签（从以下选择：温情 / 励志 / 震惊 / 搞笑 / 感动 / 愤怒）
4. 目标人群（一句话描述）

    进阶提示词¶

    你是一位资深短视频内容总监，熟悉各平台算法逻辑和用户情绪触发机制。

请分析以下内容想法，从平台适配性、情绪共鸣强度、选题竞争度三个维度进行评估，并给出差异化的选题方向。

内容想法：【粘贴你的想法】
目标平台：【平台名称】
已有参考视频（可选）：【粘贴同类视频标题或链接描述】

输出格式：
- 维度分析（三个维度逐一说明）
- 差异化选题方向（2～3 个，每个说明差异化角度）
- 最推荐方向及理由
- 情绪标签 + 目标人群

    审查提示词¶

    请检查以下选题方向是否存在以下问题：
1. 选题方向过于宽泛，无法判断具体做什么？
2. 情绪标签与内容方向是否匹配？
3. 目标人群是否过于模糊（例如"所有人"）？
4. 是否与常见爆款题材高度重叠而无差异化？

【粘贴选题判断专家的输出内容】

逐项说明问题，并给出修改建议。

    修正提示词¶

    根据以下审查意见，重新给出选题方向建议。保留原始内容想法的核心，但修正已发现的问题。

原始想法：【粘贴原始想法】
审查意见：【粘贴审查结果】

重新输出：选题方向 + 情绪标签 + 目标人群。

    验收标准：

    • [ ] 是否给出了 2 个以上可区分的选题方向？
    • [ ] 情绪标签是否具体（不是"好内容"这类废话）？
    • [ ] 目标人群是否具体到"某类人在某种情境下"？
    • [ ] 传播潜力评估是否给出了理由，而不只是结论？

    节点 2：故事选取专家¶

    职责：在选题方向确定后，从多个可能的故事角度中选出最适合视频化的一个，并给出改编方向。

    输入：节点 1 的选题方向 + 情绪标签 + 目标人群

    输出：选定的故事角度 + 选择理由 + 改编方向（3 句话以内）

    基础提示词¶

    你是一位短视频故事策划师。请根据以下选题方向，提出 3 个可能的故事角度，并推荐最适合短视频的一个。

选题方向：【粘贴节点 1 的推荐方向】
情绪标签：【粘贴情绪标签】
目标人群：【粘贴目标人群】

对每个故事角度说明：人物是谁、核心事件是什么、最打动人的瞬间是什么。
推荐理由：为什么这个角度最适合短视频？

    进阶提示词¶

    你是一位短视频故事总监，专注于情绪驱动型内容。请根据以下选题信息，提出 3 个故事角度，并从视频化难度、情绪强度、画面感三个维度对每个角度打分（1～5 分），最终推荐得分最高的角度。

选题方向：【粘贴选题方向】
情绪标签：【粘贴情绪标签】
视频时长限制：【粘贴时长】

输出：3 个角度的评分表 + 推荐角度 + 改编方向（如何调整使其更适合视频化）

    审查提示词¶

    请检查以下故事角度是否存在以下问题：
1. 故事角度是否太平淡，缺少冲突或转折点？
2. 人物是否有明确动机？
3. 这个故事在 60 秒内讲完是否可行？
4. 改编方向是否具体可执行？

【粘贴故事选取专家的输出内容】

逐项说明问题。

    修正提示词¶

    根据审查意见，重新调整故事角度和改编方向。保留原始故事的核心元素，但增强冲突感和视频化可行性。

原始故事角度：【粘贴原始角度】
审查意见：【粘贴审查结果】

    验收标准：

    • [ ] 推荐的故事角度是否有明确的人物和事件？
    • [ ] 改编方向是否在 3 句话以内说清楚了"怎么改"？
    • [ ] 这个故事是否在目标时长内可以讲完？

    节点 3：故事扩写专家¶

    职责：将选定的故事角度扩写成有结构的完整故事文本，包含前三秒钩子、冲突、转折和结尾。

    输入：节点 2 的选定故事角度 + 改编方向 + 情绪标签

    输出：完整故事文本（400～800 字，视目标时长而定）

    基础提示词¶

    请根据我提供的故事角度，扩写成一个适合短视频的完整故事。

故事角度：【粘贴节点 2 的推荐角度】
目标时长：【粘贴时长】
情绪标签：【粘贴情绪标签】

要求：
- 第一句话必须是强钩子（让人想继续看）
- 有明确的人物、目标、阻碍和转折
- 结尾有情绪余味，不能是平淡结局
- 字数控制在【目标时长对应字数】以内

    进阶提示词¶

    你是一位短视频故事专家，擅长用情绪驱动叙事。请根据以下信息，扩写成一个画面感强、节奏紧凑的短视频故事。

故事角度：【粘贴故事角度】
情绪标签：【粘贴情绪标签】
目标人群：【粘贴目标人群】
目标时长：【粘贴时长】

结构要求：
- 前 3 秒：强钩子（用一句话制造悬念或情绪冲击）
- 中间段：建立人物动机 → 出现阻碍 → 冲突升级
- 转折点：意料之外但情理之中的转变
- 结尾：情绪高点或留有余味的收尾

输出：完整故事文本 + 每段的情绪标注（括号内说明该段的情绪作用）

    审查提示词¶

    请检查以下短视频故事是否存在以下问题，逐项说明：

1. 前三秒是否有足够强的钩子？
2. 人物动机是否在前 1/3 内交代清楚？
3. 是否有明显的冲突转折点？
4. 结尾是否有情绪余味（而不是平淡收场）？
5. 画面感是否足够（每个段落是否能想象出具体画面）？
6. 字数是否超过目标时长对应的字数？

【粘贴故事文本】
目标时长：【粘贴时长】

逐项输出：问题描述 + 修改建议。

    修正提示词¶

    根据以下审查意见，重写这个故事。保留原始主题和人物，但修正指出的问题。

原始故事：【粘贴原始故事】
审查意见：【粘贴审查结果】

重点修正：前三秒钩子、人物动机清晰度、冲突强度、结尾余味。

    验收标准：

    • [ ] 第一句话是否让人想继续读下去？
    • [ ] 故事中是否有明确的冲突和转折？
    • [ ] 结尾是否有情绪收尾而不是流水账式结束？
    • [ ] 字数是否在目标时长范围内？

    节点 4：脚本生成专家¶

    职责：将故事文本转化为可拍摄的分段脚本，每段包含台词、画面描述和时长建议。

    输入：节点 3 的完整故事文本 + 目标时长

    输出：分段脚本表格（段落编号、台词、画面描述、时长）

    基础提示词¶

    请将以下故事文本转化为短视频脚本。

故事文本：【粘贴完整故事】
目标时长：【粘贴时长】

请按以下格式输出每一段：
| 段落 | 台词（旁白或对话） | 画面描述 | 建议时长 |
|------|----------------|---------|---------|

    进阶提示词¶

    你是一位短视频导演，擅长把故事文本转化为可执行的拍摄脚本。

请将以下故事转化为分段脚本，注意：
- 每段台词不超过 20 字（口播适应速度）
- 画面描述要具体到"谁在哪里做什么"
- 情绪转折段落要标注"情绪：XXX"
- 最后一列注明该段的叙事功能（钩子/建立/冲突/转折/结尾）

故事文本：【粘贴故事】
目标时长：【粘贴时长】

    审查提示词¶

    请检查以下脚本是否存在以下问题：
1. 台词是否有口播过长（单句超过 20 字）的情况？
2. 画面描述是否具体（"模糊的场景"不算）？
3. 各段时长加总是否与目标时长匹配？
4. 是否有段落缺少视觉内容（只有台词没有画面）？

【粘贴脚本内容】

逐项说明问题。

    修正提示词¶

    根据以下审查意见，修改脚本。不要改变故事结构，只修正指出的具体问题。

原始脚本：【粘贴原始脚本】
审查意见：【粘贴审查结果】

    验收标准：

    • [ ] 脚本是否按表格格式输出，台词和画面分开？
    • [ ] 每段台词是否在 20 字以内？
    • [ ] 所有段落时长加总是否与目标时长大致吻合？
    • [ ] 每段是否都有具体的画面描述？

    节点 5：分镜设计专家¶

    职责：将脚本每个段落转化为具体的镜头语言描述，包括景别、运动方式和画面构成。

    输入：节点 4 的分段脚本

    输出：分镜表（镜号、景别、镜头运动、画面内容、时长）

    基础提示词¶

    请将以下短视频脚本转化为分镜表。

脚本：【粘贴分段脚本】

每个分镜请包含：
- 镜号
- 景别（远景/全景/中景/近景/特写）
- 镜头运动（固定/推/拉/横移/跟拍/航拍）
- 画面内容（具体描述画面里有什么）
- 时长（秒）

    进阶提示词¶

    你是一位专业分镜师，擅长为情感类短视频设计有节奏感的镜头语言。

请将以下脚本转化为分镜设计方案。要求：
- 情绪高潮段落使用特写或推镜头强化情绪
- 叙事段落使用全景或中景建立场景
- 每个分镜说明构图重点（主体在哪里，背景是什么）
- 注明相邻镜头的切换方式（硬切/淡入淡出/转场）

脚本：【粘贴分段脚本】
风格参考：【填入风格，例如：电影感/纪录片风/手机竖拍风】

    审查提示词¶

    请检查以下分镜设计是否存在以下问题：
1. 是否有连续多个相同景别（缺少变化）？
2. 情绪高潮是否用了对应的镜头语言强化？
3. 是否有分镜画面内容描述过于模糊？
4. 总时长是否与脚本时长匹配？

    常见错误¶

    错误 1：跳过审查直接进下一个节点

    表现：脚本问题带入分镜，分镜问题带入提示词，最终提示词质量低劣。

    修复：每个节点完成后，必须先运行"审查提示词"，确认无问题再进入下一节点。

    错误 2：图像提示词中人物描述不一致

    表现：同一个人物在不同分镜里外貌和服装发生变化，AI 生成结果"换脸换衣"。

    修复：在开始节点 6 之前，先确定一段固定的"角色描述词"，所有分镜的提示词都必须包含这段描述词。

    错误 3：视频提示词描述静止画面

    表现：提示词只描述画面构成，没有说明运动过程，导致生成的视频像幻灯片。

    修复：视频提示词必须描述从画面开始到结束的运动变化，例如"从远景缓慢推进到近景"。

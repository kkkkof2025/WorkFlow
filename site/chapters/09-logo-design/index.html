    工作流 09：从品牌背景到可落地 Logo 方向的 AI 多专家工作流¶

    适用岗位：品牌设计师、企业宣传人员、创业者、视觉设计师

    真实场景：你是一家新成立的中餐厅老板，或者刚拿到融资的创业团队，需要设计一个 Logo，但你不知道从哪里开始，也不知道怎么把"有辨识度、有中国味、能注册商标"这些模糊要求转化为设计师可以执行的方向。

    最终目标：一份完整的 Logo 创意提案，包含：品牌关键词提炼、视觉方向建议（3个）、汉字变形方案（如适用）、图形结构说明、色彩策略、应用场景清单、原创性风险评估，以及可以直接交给设计师或图像生成工具继续使用的提示词描述。

    输入材料清单¶

    • 品牌基本信息（品牌名称、行业、成立背景）
    • 目标受众（年龄、消费偏好、使用场景）
    • 品牌定位（高端/亲民/专业/创意/传统）
    • 竞品 Logo 参考（可选，描述竞品的视觉特点）
    • 禁止方向（不希望出现的风格或颜色）
    • 使用场景（门头/名片/APP图标/包装）

    工作流总览¶

    节点 1：品牌信息解析专家
  ↓ 输出：品牌性格词 + 行业视觉惯例 + 差异化机会
节点 2：品牌关键词提炼专家
  ↓ 输出：2-3个可视化设计关键词
节点 3：视觉隐喻专家
  ↓ 输出：3个视觉方向提案（含隐喻逻辑）
节点 4：汉字变形专家（招牌节点）
  ↓ 输出：汉字变形方案 + 执行指引 + AI提示词
节点 5：图形结构专家
  ↓ 输出：图形结构方案 + 各版本说明
节点 6：色彩策略专家
  ↓ 输出：主色方案 + 完整色值规范
节点 7：应用场景专家
  ↓ 输出：各场景适配规范
节点 8：原创性与可注册性审查专家
  ↓ 输出：风险评估报告
节点 9：Logo 提案生成专家
  ↓ 输出：完整 Logo 创意提案书
最终交付：可落地的 Logo 创意提案

    专家节点详解¶

    C 自查审稿版和 D 返修优化版是当前节点内部的工作模式，不是新的专家角色。

    节点 1：品牌信息解析专家¶

    1.1 节点定位¶

    从品牌背景中提取与 Logo 设计直接相关的核心要素，建立品牌信息框架。很多人直接说"设计一个 Logo"，但没有先整理品牌信息，导致设计师反复返稿——这个节点解决的是"说清楚品牌是什么"的问题。

    1.2 输入与输出¶

    输入：品牌基本信息 + 目标受众 + 品牌定位 + 禁止方向

    输出：品牌信息框架（品牌性格词 + 受众视觉偏好 + 行业视觉惯例 + 差异化机会 + 视觉禁忌）

    1.3 使用顺序¶

    1. 先用「快速生成版」得到品牌信息框架初稿。
    2. 有复杂品牌背景或细分行业时，改用「专家增强版」。
    3. 用「自查审稿版」检查是否遗漏了行业视觉惯例。
    4. 有问题则用「返修优化版」补充。
    5. 对照 1.6 验收清单确认，通过后交给节点 2。

    1.4 提示词包¶

    A. 快速生成版¶
    你是一位品牌顾问。请根据以下品牌信息，梳理 Logo 设计需要考虑的核心要素。

品牌名称：【填入】
行业：【填入】
目标受众：【填入】
品牌定位：【填入】
禁止方向：【填入】

输出：
1. 品牌性格关键词（3-5个，例如：专业/亲切/创新/传统/活泼）
2. 目标受众的视觉偏好（他们喜欢什么风格的 Logo）
3. 行业常见 Logo 视觉惯例（这个行业的 Logo 通常是什么样的）
4. 差异化机会（如何在符合行业认知的同时与竞品区分）
5. 视觉禁忌（明确不能出现的元素，结合用户禁止方向和行业暗雷）
    B. 专家增强版¶
    你是一位资深品牌战略顾问，拥有分析 Logo 设计需求的系统方法。

任务：对以下品牌信息进行深度解析，为后续 Logo 视觉方向提供品牌层面的依据。

输入：
- 品牌基本信息：【填入所有品牌资料】
- 竞品参考（可选）：【填入竞品视觉描述】

解析维度：
1. 品牌核心承诺（给消费者的核心价值，一句话）
2. 品牌性格（3-5个形容词，要有可视化对应，例如"精准"→细线条/锐角）
3. 目标受众视觉解码（该群体喜欢什么视觉语言，不喜欢什么）
4. 行业视觉惯例分析（同行业 Logo 常见的元素/颜色/字体风格，列举3个规律）
5. 差异化方向（在哪个视觉维度可以做到不同）
6. 视觉禁忌（包括用户提供的禁止方向和行业暗雷）

约束：所有分析必须指向可视化的设计方向，不能只是品牌价值的文字描述。

输出：品牌信息框架（6个维度）+ 对 Logo 设计的具体指导建议
    C. 自查审稿版¶

    此为当前节点的自查模式，不是新的专家角色。

    请检查以下品牌信息框架是否存在以下问题：
1. 品牌性格关键词是否能直接对应视觉元素（"有内涵"不能直接设计，"精准细线"可以）？
2. 行业视觉惯例分析是否基于真实的行业案例，还是泛泛而谈？
3. 差异化方向是否具体到设计维度，而非只说"要有个性"？

【粘贴品牌信息框架内容】

逐条说明问题。
    D. 返修优化版¶
    根据自查意见修改品牌信息框架。将抽象性格词替换为可视化元素对应，补充具体行业案例，明确差异化的具体维度。

原始框架：【粘贴原始内容】
自查意见：【粘贴自查结果】

    1.5 交付给下游节点¶

    将品牌信息框架完整复制，交给节点 2（品牌关键词提炼专家）。

    1.6 人工验收清单¶

    • [ ] 品牌性格关键词是否都能对应到具体视觉元素？
    • [ ] 是否明确列出了至少 3 条行业视觉惯例？
    • [ ] 差异化方向是否具体到设计维度？

    节点 2：品牌关键词提炼专家¶

    2.1 节点定位¶

    从品牌信息框架中提炼出 2-3 个可以驱动视觉设计的核心关键词，作为后续视觉隐喻和图形设计的出发点。关键词不是广告语，而是设计语言——"速度"可以对应斜线和动感，"守护"可以对应圆形和包围结构。

    2.2 输入与输出¶

    输入：节点 1 的品牌信息框架

    输出：2-3 个品牌设计关键词 + 每个关键词的视觉化方向说明

    2.3 使用顺序¶

    1. 先用「快速生成版」得到关键词初稿。
    2. 关键词过多或视觉方向重叠时，改用「专家增强版」做筛选。
    3. 用「自查审稿版」检查关键词是否独特且可视化。
    4. 有问题则用「返修优化版」精炼。
    5. 对照 2.6 验收清单确认，通过后交给节点 3 和节点 4。

    2.4 提示词包¶

    A. 快速生成版¶
    你是一位品牌视觉策划师。请从以下品牌信息框架中提炼 2-3 个核心设计关键词。

品牌信息框架：【粘贴节点1的输出】

对每个关键词输出：
1. 关键词（1-2个汉字或英文单词）
2. 与品牌的关联逻辑
3. 视觉化方向（可对应的图形语言：形状/线条/结构）
    B. 专家增强版¶
    你是一位资深品牌视觉策略师，擅长从品牌内涵中提炼可落地的设计关键词。

任务：从以下品牌信息框架中精炼出2-3个设计关键词，每个词必须既反映品牌内涵又能直接指导视觉设计。

输入：
- 品牌信息框架：【粘贴节点1的完整输出】

关键词筛选标准：

    节点 4：汉字变形专家（招牌节点）¶

    4.1 节点定位¶

    这是本工作流的招牌节点。 专门处理中文品牌名称的字形设计，解决以下五个核心问题：

    1. 哪个字适合变形：不是所有汉字都值得变形，复杂结构的字变形后更难识别
    2. 保留哪些关键笔画：有些笔画是字的骨架，去掉就不认识了
    3. 哪些笔画可以与行业符号融合：哪一笔可以被箭头、工具、自然符号替代
    4. 如何避免字形变得不可识别：变形的边界在哪里
    5. 如何输出给设计师或 AI 模型使用：给出可执行的笔画级指引和英文提示词

    汉字变形不是把字拉伸或加装饰，而是在保证识别度的前提下，用字形本身传达品牌含义。

    4.2 输入与输出¶

    输入：节点 2 的品牌关键词 + 节点 3 的视觉隐喻方案（图文或纯文字方向）+ 品牌名称

    输出：汉字变形方案说明（逐字分析 + 变形策略 + 骨架保留清单 + 行业符号融合方案 + 执行指引 + AI提示词）

    4.3 使用顺序¶

    1. 先用「快速生成版」得到汉字变形思路初稿。
    2. 有复杂字形或多字品牌名时，必须使用「专家增强版」。
    3. 用「自查审稿版」检查变形后是否仍可识别、骨架保留是否明确。
    4. 有问题则用「返修优化版」修正。

    常见错误¶

    错误 1：直接从"做一个 Logo"开始，跳过品牌信息解析

    表现：做了 3 版 Logo 都被否掉，原因是没有先明确品牌关键词和视觉方向，导致设计师和客户对"好 Logo"的理解不一致。

    修复：必须先完成节点 1 和节点 2，在双方对品牌关键词和视觉方向达成一致后，再开始任何视觉设计工作。

    错误 2：汉字变形过度导致不可识别

    表现：把品牌名称里某个汉字做了大量变形，让人完全看不出原来的字，变成了"有文字感的图形"。

    修复：执行节点 4 的骨架保留分析，明确哪些笔画是该字的最小识别单元——这些骨架笔画绝对不能变形或移除。行业符号融合比例控制在字形面积 40% 以内。

    错误 3：设计完成才发现风险

    表现：Logo 完成后，在商标检索时被发现与某个品牌高度相似，或者注册时被驳回。

    修复：执行节点 8，在提案阶段就做初步原创性评估。发现风险时还有时间修改方案，比完成设计后再改成本低得多。

    人工验收清单¶

    • [ ] 提案书是否包含三个视觉语言明显不同的设计方向？
    • [ ] 汉字变形方案（如有）是否明确了不能变形的骨架笔画清单？
    • [ ] 色彩方案是否包含完整四套色值（RGB/HEX/CMYK/Pantone）？
    • [ ] 原创性和商标可注册性风险是否已评估？
    • [ ] AI 工具参考图生成提示词是否可以直接复制使用（英文）？

    延伸玩法¶

    • 变体 1：纯英文品牌版：跳过节点 4（汉字变形），将节点 3 的图文结合方向聚焦于英文字母变形，用同样的分析框架处理英文字母的笔画变形和行业符号融合。
    • 变体 2：子品牌扩展：主品牌 Logo 完成后，用本工作流的节点 5-7 为子品牌或产品线生成风格一致但有区分度的衍生 Logo 方案。
    • 进阶组合：将本工作流输出的 AI 参考图提示词（节点 9）输入"工作流 07（图像生成工作流）"的节点 6，使用图像生成工具快速产出 Logo 参考图，加速设计决策。

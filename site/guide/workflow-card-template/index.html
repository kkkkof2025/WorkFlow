    专家节点详解¶

    每个节点包含六部分：节点定位、输入与输出、使用顺序、提示词包（A 快速 / B 专家 / C 自查 / D 返修）、交付给下游节点、人工验收清单。

    C 自查审稿版和 D 返修优化版是当前节点内部的工作模式，不是新的专家角色。

    节点 1：【节点名称】¶

    1.1 节点定位¶

    【说明这个专家节点在整个工作流中的作用，以及为何需要一个专门节点处理这项任务。不超过 3 句话。】

    1.2 输入与输出¶

    输入：【接收什么材料，从哪个节点或来源获得】

    输出：【产出什么，格式是什么】

    1.3 使用顺序¶

    1. 先用「快速生成版」产出初稿。
    2. 再用「专家增强版」提升质量（可选，适合要求更高的场景）。
    3. 用「自查审稿版」检查当前节点的输出质量。
    4. 如有问题，用「返修优化版」根据自查结果修改。
    5. 人工对照 1.6 验收清单确认，通过后交给下游节点。

    1.4 提示词包¶

    A. 快速生成版¶
    【直接可用的提示词，50-100 字，角色 + 任务 + 输出格式清晰，不留无法识别的占位符】
    B. 专家增强版¶
    【高质量版，100-200 字，必须包含：角色设定、具体任务、输入说明、执行步骤或约束、输出格式要求、质量标准】
    C. 自查审稿版¶

    此为当前节点的自查模式，不是新的专家角色。

    【列出 3-5 个具体可判断的审查问题，每条一行，格式：是否存在 X 问题？】
【粘贴当前节点的输出内容】
逐条说明问题。
    D. 返修优化版¶
    【根据自查结果返修，明确"保留什么、修正什么"，50-100 字】
原始输出：【粘贴原始输出】
自查意见：【粘贴自查结果】

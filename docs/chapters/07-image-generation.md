# 工作流 07：从创意需求到稳定图像提示词的 AI 多专家工作流

**适用岗位**：设计师、自媒体创作者、品牌运营、短视频创作者、活动宣传人员

**真实场景**：你需要为品牌活动生成一批视觉一致的宣传图，或者为短视频制作系列封面图，但每次生成的图像风格不统一、人物换脸、背景乱飘——你需要一套可以稳定复用的图像提示词体系。

**最终目标**：一套可稳定复用的图像提示词体系，包含：视觉风格定义、主体描述词库、一致性控制词组、负面提示词、出图审查标准，以及适用于不同场景的提示词变体。

---

## 输入材料清单

- 创意需求描述（一句话或一段话，说明要生成什么）
- 使用场景（宣传海报 / 系列封面 / 故事插图 / 产品展示）
- 目标工具（Midjourney v6 / Stable Diffusion / 即梦 / DALL-E）
- 参考风格（可选，1-3 张参考图的文字描述，或风格关键词）
- 画幅要求（16:9 / 9:16 / 1:1 / 4:3）

---

## 工作流总览

```
节点 1：需求解析专家
  ↓ 输出：创意拆解清单（主体 + 场景 + 情绪 + 用途）
节点 2：视觉风格定位专家
  ↓ 输出：风格关键词组 + 参考画风说明
节点 3：主体元素设计专家
  ↓ 输出：主体描述词组（人物 / 物体 / 场景的具体描述）
节点 4：构图与镜头语言专家
  ↓ 输出：构图方案 + 景别 + 视角 + 焦距描述
节点 5：光影色彩专家
  ↓ 输出：光线方案 + 配色策略 + 情绪色调词组
节点 6：图像提示词生成专家
  ↓ 输出：完整可用英文提示词（结构化）
节点 7：一致性控制专家（招牌节点）
  ↓ 输出：一致性词库 + 系列图复用方案
节点 8：负面提示词专家
  ↓ 输出：针对性负面提示词组
节点 9：出图审查与返修专家
  ↓ 输出：审查报告 + 修正后提示词
最终交付：可稳定复用的图像提示词体系
```

---

## 专家节点详解

每个节点包含六部分：节点定位、输入与输出、使用顺序、提示词包（A 快速 / B 专家 / C 自查 / D 返修）、交付给下游节点、人工验收清单。

**C 自查审稿版和 D 返修优化版是当前节点内部的工作模式，不是新的专家角色。**

---

### 节点 1：需求解析专家

#### 1.1 节点定位

将模糊的创意需求拆解为图像生成所需的结构化要素，防止在生成时因描述笼统导致 AI 理解偏差。"一张好看的海报"和"深色背景、单人、科技感、竖版海报"之间的差距，就是这个节点解决的问题。

#### 1.2 输入与输出

**输入**：原始创意需求描述 + 使用场景说明

**输出**：创意拆解清单（主体对象 + 场景环境 + 情绪基调 + 用途约束 + 禁止元素）

#### 1.3 使用顺序

1. 先用「快速生成版」得到拆解清单初稿。
2. 需求较复杂或多人物场景时，改用「专家增强版」。
3. 用「自查审稿版」检查拆解是否完整、是否有遗漏约束。
4. 有问题则用「返修优化版」补充。
5. 对照 1.6 验收清单确认，通过后交给节点 2。

#### 1.4 提示词包

##### A. 快速生成版

```
你是一位图像需求分析师。请将以下创意需求拆解为图像生成所需的结构化要素。

需求描述：【粘贴创意需求】
使用场景：【填入：宣传海报 / 系列封面 / 插图 / 产品展示】

输出：
1. 主体对象（画面中心是什么）
2. 场景环境（背景、地点、时间）
3. 情绪基调（暖/冷/中性，情绪关键词）
4. 用途约束（竖版/横版，有无文字区域，有无品牌色要求）
5. 禁止元素（哪些不能出现，例如：不能有文字水印、不能有多余人物）
```

##### B. 专家增强版

```
你是一位资深视觉创意总监，专注于将模糊创意需求转化为精准的图像生成指令。

任务：对以下创意需求进行系统性拆解，输出可直接指导后续节点的结构化需求清单。

输入：
- 创意需求：【粘贴需求描述】
- 使用场景：【填入场景】
- 目标工具：【填入 AI 绘图工具名称】

拆解维度：
1. 主体层（Who/What）：画面中心对象的身份、动作、数量
2. 场景层（Where/When）：地点类型、时间段、天气/氛围
3. 情绪层（How it feels）：情绪关键词 + 视觉情绪对应（暖色/冷色/高对比/低饱和）
4. 功能层（Usage constraints）：版式要求、留白需求、品牌色限制、禁止内容
5. 差异化层（What makes it unique）：这张图与同类图的区别是什么

约束：禁止输出无法视觉化的需求（例如"有内涵"不算需求）；所有元素必须是可以在画面中呈现的视觉内容。

输出：5个维度的结构化需求清单 + 优先级标注（核心要素 vs 加分要素）
```

##### C. 自查审稿版

> 此为当前节点的自查模式，不是新的专家角色。

```
请检查以下需求拆解清单是否存在以下问题：
1. 是否有无法视觉化的描述（例如"有品味""高端感"，未说明具体视觉特征）？
2. 主体对象描述是否精确到数量和状态（"人"不够，需要"一位站立的青年女性"）？
3. 用途约束是否完整（版式比例、留白位置是否说明）？
4. 是否遗漏了禁止元素（例如：需要配文字但忘了说明留白区域位置）？

【粘贴需求拆解清单】

逐条说明问题。
```

##### D. 返修优化版

```
根据自查意见修改需求拆解清单。将抽象描述替换为可视化元素，补充缺失的数量/状态信息，完善用途约束和禁止元素。

原始清单：【粘贴原始清单】
自查意见：【粘贴自查结果】
```

#### 1.5 交付给下游节点

将结构化需求清单完整复制，交给节点 2（视觉风格定位专家）。节点 2 将基于情绪基调和功能约束，确定图像的视觉风格方向。

#### 1.6 人工验收清单

- [ ] 主体对象描述是否精确到数量、状态和关键特征？
- [ ] 是否有明确的版式要求（横版/竖版/比例）？
- [ ] 是否列出了至少 1 条禁止元素？

### 节点 2：视觉风格定位专家

#### 2.1 节点定位

基于需求拆解结果，确定图像的视觉风格方向，输出可直接用于提示词的风格关键词组。同样是"科技感"，赛博朋克风和极简科技白风的提示词完全不同——这个节点负责明确化。

#### 2.2 输入与输出

**输入**：节点 1 的结构化需求清单（情绪层 + 功能层）

**输出**：风格类型标签（2-3 个）+ 主风格关键词组（可直接用于提示词）+ 禁用风格说明

#### 2.3 使用顺序

1. 先用「快速生成版」得到风格关键词初稿。
2. 需要在多个风格方向中选择时，改用「专家增强版」做对比分析。
3. 用「自查审稿版」检查风格词是否与需求匹配、是否是目标工具认识的词汇。
4. 有问题则用「返修优化版」修正。
5. 对照 2.6 验收清单确认，通过后交给节点 3。

#### 2.4 提示词包

##### A. 快速生成版

```
你是一位视觉风格顾问。请根据以下需求清单，确定图像的视觉风格方向。

需求清单：【粘贴节点1的输出】
目标工具：【填入工具名称】

输出：
1. 风格类型标签（2-3个，例如：电影级写实 / 日系扁平插画 / 深色科幻概念图）
2. 主风格英文关键词组（可直接加入提示词，10-15个词）
3. 禁用风格说明（哪些风格与需求不符，需要排除）
```

##### B. 专家增强版

```
你是一位资深视觉风格总监，熟悉 Midjourney/SD 的风格关键词效果。

任务：基于以下需求清单，提供 3 个可选风格方向，并推荐最适合当前用途的方向。

输入：
- 需求清单：【粘贴节点1的完整输出】
- 目标工具：【填入工具名称】

每个风格方向输出：
- 风格名称（具体到亚类型，例如：Wes Anderson 电影色调，而不只是"电影风格"）
- 英文关键词组（10-15个，可直接用于提示词）
- 适用场景说明（为什么这个风格适合当前需求）
- 潜在问题（使用这个风格时需要注意什么）

推荐理由：最终推荐哪个方向，说明与需求的匹配度。

约束：关键词必须是目标工具实际支持的词汇，不能自造词；中文需求必须翻译为英文关键词。

输出：3个风格方向对比 + 最终推荐 + 推荐英文关键词组
```

##### C. 自查审稿版

> 此为当前节点的自查模式，不是新的专家角色。

```
请检查以下风格定位是否存在以下问题：
1. 风格关键词是否是目标 AI 工具能够识别的标准词汇（不能是自造词或中文词）？
2. 风格方向是否与需求清单中的情绪基调和用途约束一致？
3. 是否有风格词与需求中的禁止元素产生冲突？
4. 关键词数量是否在合理范围（过多会稀释权重，建议不超过 20 个）？

【粘贴风格定位内容】

逐条说明问题。
```

##### D. 返修优化版

```
根据自查意见修正风格定位。替换自造词/中文词为标准英文关键词，修正与需求冲突的风格词，精简超量关键词。

原始风格词组：【粘贴原始风格定位】
自查意见：【粘贴自查结果】
目标工具：【填入工具名称】
```

#### 2.5 交付给下游节点

将推荐风格方向的英文关键词组复制，交给节点 3（主体元素设计专家）和节点 5（光影色彩专家）。风格词会融入最终的完整提示词中。

#### 2.6 人工验收清单

- [ ] 风格关键词是否都是英文，且是目标工具支持的标准词汇？
- [ ] 风格方向是否与需求的情绪基调一致？
- [ ] 关键词总数是否不超过 20 个？

### 节点 3：主体元素设计专家

#### 3.1 节点定位

设计画面主体对象的详细视觉描述，包括人物外貌、物体特征或场景细节。主体描述是提示词中权重最高的部分——描述越精确，生成结果越稳定。

#### 3.2 输入与输出

**输入**：节点 1 的主体层描述 + 节点 2 的风格关键词组

**输出**：主体元素英文描述词组（按优先级排列）

#### 3.3 使用顺序

1. 先用「快速生成版」得到主体描述初稿。
2. 有人物外貌或复杂物体时，改用「专家增强版」。
3. 用「自查审稿版」检查描述是否精确、是否与风格一致。
4. 有问题则用「返修优化版」修正。
5. 对照 3.6 验收清单确认，通过后交给节点 4。

#### 3.4 提示词包

##### A. 快速生成版

```
你是一位 AI 图像提示词工程师。请根据以下主体需求，生成主体元素描述词组。

主体对象：【粘贴节点1的主体层描述】
视觉风格：【粘贴节点2的风格关键词组】

输出：
主体英文描述词组（按重要性排列，最重要的放最前面）
格式：[主体身份/类型], [外貌/状态描述], [动作/姿势], [与场景的关系]
```

##### B. 专家增强版

```
你是一位专注于 AI 图像提示词优化的视觉描述专家。

任务：为以下主体对象生成精确的英文描述词组，确保 AI 能准确理解并生成预期画面。

输入：
- 主体对象需求：【粘贴节点1的主体层描述】
- 风格方向：【粘贴节点2的风格关键词组】

描述层次：
1. 身份层：主体是什么（人/物/场景，精确分类）
2. 外观层：关键外观特征（对人物：性别、年龄区间、肤色、发型；对物体：形状、材质、颜色）
3. 状态层：当前动作或姿势（用动词 + 副词精确描述）
4. 关系层：主体与场景的空间关系（前景/中景，朝向，距离）

约束：
- 所有描述词必须是英文，且 AI 工具能够理解
- 人物描述不能只写"beautiful woman"，需要精确到面部特征和体型
- 避免使用含义模糊的词汇（如 nice、good、amazing）

输出：按优先级排列的主体描述英文词组（每层 3-5 个关键词，用逗号分隔）
```

##### C. 自查审稿版

> 此为当前节点的自查模式，不是新的专家角色。

```
请检查以下主体描述词组是否存在以下问题：
1. 是否有含义模糊的词汇（beautiful、nice、amazing 等无法精确生成的词）？
2. 人物描述是否过于简略（如只写"a woman"，缺少年龄、外貌、状态信息）？
3. 描述词是否与风格关键词产生矛盾（例如极简风格里有复杂装饰描述）？
4. 动作/姿势描述是否清晰可生成（"思考中"不清晰，"hand on chin, looking down"清晰）？

【粘贴主体描述词组】

逐条说明问题。
```

##### D. 返修优化版

```
根据自查意见修改主体描述词组。替换模糊词汇，补充缺失的外貌细节，消除风格矛盾，明确动作描述。

原始词组：【粘贴原始主体描述】
自查意见：【粘贴自查结果】
```

#### 3.5 交付给下游节点

将主体描述词组按优先级排列后复制，交给节点 4（构图与镜头语言专家）。构图方案需要根据主体特征来设计。

#### 3.6 人工验收清单

- [ ] 主体描述是否精确到关键外观特征（不只是"人"或"物体"）？
- [ ] 动作/姿势描述是否可以被 AI 工具准确理解和生成？
- [ ] 描述词是否都是英文且无歧义？

---

### 节点 4：构图与镜头语言专家

#### 4.1 节点定位

确定画面的构图方式、景别和视角，为最终提示词提供构图参数。构图决定了观者视线的引导方式——同样的主体，特写和全景传达的信息完全不同。

#### 4.2 输入与输出

**输入**：节点 1 的用途约束 + 节点 3 的主体描述

**输出**：构图方案描述（景别 + 视角 + 主体位置 + 画幅参数）

#### 4.3 使用顺序

1. 先用「快速生成版」得到构图方案初稿。
2. 有特殊叙事需求或多主体时，改用「专家增强版」。
3. 用「自查审稿版」检查构图是否与用途匹配。
4. 有问题则用「返修优化版」调整。
5. 对照 4.6 验收清单确认，通过后交给节点 5。

#### 4.4 提示词包

##### A. 快速生成版

```
你是一位摄影构图师。请根据以下信息，确定图像构图方案。

主体描述：【粘贴节点3的主体词组摘要】
用途约束：【粘贴节点1的用途层（版式、留白等）】

输出：
1. 景别（特写/近景/中景/全景/远景）
2. 视角（正面/侧面/俯视/仰视/鸟瞰）
3. 主体在画面中的位置（居中/三分法/左偏/右偏）
4. 对应的英文构图关键词
5. 画幅参数（--ar 对应值）
```

##### B. 专家增强版

```
你是一位兼具摄影和平面设计背景的构图专家，熟悉 AI 图像工具的构图关键词效果。

任务：为以下图像需求设计构图方案，确保构图服务于用途目标（宣传/叙事/装饰）。

输入：
- 主体对象：【粘贴节点3的主体描述】
- 使用用途：【粘贴节点1的用途约束】
- 风格方向：【粘贴节点2的风格标签】

构图设计维度：
1. 景别选择及原因（为什么这个景别适合当前用途）
2. 视角选择及原因（平拍/仰拍/俯拍对情绪的影响）
3. 三分法 / 对称 / 对角线等构图原则的应用
4. 留白策略（哪个方向留白，为文字或 Logo 预留空间）
5. 对应英文关键词（lens description + composition terms）

约束：如果用途是宣传海报，必须考虑文字覆盖区域；如果是系列图，所有图的景别必须统一。

输出：构图方案说明 + 英文关键词 + 画幅参数建议
```

##### C. 自查审稿版

> 此为当前节点的自查模式，不是新的专家角色。

```
请检查以下构图方案是否存在以下问题：
1. 构图是否与使用用途一致（宣传海报一般需要留白区域，但方案中是否考虑到了）？
2. 景别选择是否适合主体类型（人物特写不适合强调场景；全景不适合产品细节展示）？
3. 如果是系列图，是否规定了统一的景别和视角？

【粘贴构图方案内容】

逐条说明问题。
```

##### D. 返修优化版

### 节点 9：出图审查与返修专家

#### 9.1 节点定位

**这是一个跨内容的全局审查节点**，针对实际出图结果进行审查，判断是否需要调整提示词，以及如何调整才能解决具体问题。提示词是假设，出图结果才是事实——这个节点将两者连接起来。

#### 9.2 输入与输出

**输入**：节点 6 完整提示词 + 节点 7 一致性词库 + 节点 8 负面提示词 + 实际出图结果描述（人工填写）

**输出**：出图问题诊断报告 + 修正后提示词

#### 9.3 使用顺序

1. 先用「快速生成版」诊断最常见的出图问题。
2. 出图问题复杂或涉及一致性问题时，改用「专家增强版」。
3. 用「自查审稿版」确认修正方向是否有效。
4. 有问题则用「返修优化版」做第二轮修正。
5. 对照 9.6 验收清单确认。

## 常见错误

**错误 1：主体描述放在风格词之后**

表现：提示词以 cinematic style, 8k resolution 开头，主体描述放在第 5 个词之后，导致 AI 优先处理风格而对主体理解偏差。

修复：主体核心词（身份 + 关键外貌）必须是提示词最前面的 3 个词，风格词放在中段。

**错误 2：一致性词库建立后不完整复制**

表现：系列图中某几张忘记粘贴快速调用前缀，或只粘贴了一部分，导致人物外貌在那几张图中发生变化。

修复：建立快速调用前缀后，每次生成新图时必须完整复制前缀并粘贴到提示词最前面，一个词都不能省略。

**错误 3：负面提示词与正面提示词冲突**

表现：想生成长发角色，但负面词里出现了 long hair（可能是复制粘贴时的失误），导致 AI 生成混乱的发型结果。

修复：执行节点 8 的自查审稿版，逐字比对正面词和负面词，确认没有相同词汇同时出现在两边。

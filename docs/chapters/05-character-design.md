# 工作流 05：从故事文本到可生成角色设定图的 AI 多专家工作流

**适用岗位**：插画师、漫画创作者、短剧美术、游戏设计师、AI 绘图创作者

**真实场景**：你已经写好了一段故事文本或短剧剧本，需要为主要角色生成可以直接用于 Midjourney/Stable Diffusion 的角色设定图，要求正面/侧面/背面三视图风格统一、不同角色之间有明显视觉区分度。

**最终目标**：每个主角的完整角色设定文档，包含：角色内核分析、外貌特征描述、服装道具设计、三视图图像提示词（英文）、风格一致性描述词库、角色差异化方案。

---

## 输入材料清单

开始前请准备以下材料：

- 故事文本或剧本片段（含角色出场的描述段落，500 字以上）
- 需要设计的角色列表（姓名 + 在故事中的基本身份）
- 目标视觉风格（写实 / 动漫 / 古风 / 赛博朋克 / 其他）
- AI 绘图工具（Midjourney v6 / Stable Diffusion / 即梦等）
- 参考风格图（可选，1-3 张风格参考图的文字描述）

---

## 工作流总览

```
节点 1：角色内核提取专家
  ↓ 输出：每个角色的内核关键词（性格+动机+身份定位）
节点 2：年龄身份校准专家
  ↓ 输出：年龄/时代/身份对外貌的影响清单
节点 3：外貌特征设计专家
  ↓ 输出：详细外貌描述（面部+体型+肤色+发型）
节点 4：服装道具设计专家
  ↓ 输出：服装设计描述 + 标志性道具
节点 5：三视图提示词专家
  ↓ 输出：正面/侧面/背面三视图英文提示词
节点 6：风格一致性专家
  ↓ 输出：角色一致性描述词库 + 跨场景复用方案
节点 7：角色差异化审查专家（跨角色全局审查）
  ↓ 输出：差异化审查报告 + 修改建议
节点 8：图像生成提示词修正专家
  ↓ 输出：修正后的完整提示词组
最终交付：每个角色的完整设定文档包
```

---

## 专家节点详解

每个节点包含六部分：节点定位、输入与输出、使用顺序、提示词包（A 快速 / B 专家 / C 自查 / D 返修）、交付给下游节点、人工验收清单。

**关于节点 7（角色差异化审查专家）**：这是一个跨角色的全局审查节点，负责检查多个角色之间是否存在外貌雷同问题，性质与其他节点的自查审稿版不同——它是比较多个角色的独立审查节点。

**其他节点的 C 自查审稿版和 D 返修优化版是当前节点内部的工作模式，不是新的专家角色。**

---

### 节点 1：角色内核提取专家

#### 1.1 节点定位

从故事文本中提取每个角色的性格特征、行为动机和身份定位，为外貌设计提供内在依据。外貌设计必须从角色内核出发——一个冷酷的反派和一个温柔的女主，外貌气质应当有本质差异，而不是"换个发色"。

#### 1.2 输入与输出

**输入**：故事文本或剧本片段 + 角色列表

**输出**：每个角色的内核关键词组（3-5个性格关键词 + 核心动机 + 身份标签 + 气质方向）

#### 1.3 使用顺序

1. 先用「快速生成版」得到每个角色的内核关键词。
2. 故事文本较长或角色关系复杂时，改用「专家增强版」。
3. 用「自查审稿版」检查关键词是否精确反映故事中的角色特质。
4. 有问题则用「返修优化版」修正。
5. 对照 1.6 验收清单确认，通过后交给节点 2。

#### 1.4 提示词包

##### A. 快速生成版

```
你是一位角色设计师。请从以下故事文本中提取每个角色的内核关键词。

故事文本：【粘贴故事文本】
角色列表：【粘贴角色姓名和基本身份】

对每个角色输出：
- 性格关键词（3-5个，例如：冷静、腹黑、外冷内热）
- 核心动机（他/她最驱动自己行动的欲望或恐惧）
- 气质方向（例如：权贵压迫感 / 邻家少女清新感 / 反社会神秘感）
```

##### B. 专家增强版

```
你是一位资深角色设计总监，擅长将文字角色转化为视觉语言。

任务：从以下故事文本中精准提取每个角色的内核特质，为后续外貌设计提供可视化方向。

输入：
- 故事文本：【粘贴故事文本】
- 角色列表：【粘贴角色姓名和基本身份】

提取维度：
1. 性格关键词（3-5个，直接影响面部表情习惯和体态）
2. 核心动机（驱动行为的根本欲望，影响角色常态表情）
3. 身份标签（社会阶层/职业/时代背景，影响服装气质）
4. 气质方向（用一个视觉比喻描述，例如"寒冬腊月的第一场雪"）
5. 视觉禁忌（该角色绝对不适合的外貌元素，例如：不能有可爱系卷发）

约束：关键词必须能直接转化为视觉描述，不能使用"有内涵""很特别"等无法可视化的词汇。

输出：每个角色的内核提取表格（含5个维度）
```

##### C. 自查审稿版

> 此为当前节点的自查模式，不是新的专家角色。

```
请检查以下角色内核提取是否存在以下问题：
1. 性格关键词是否能转化为视觉元素（不能包含无法画出来的词汇）？
2. 气质方向描述是否足够具体，能给设计师明确的视觉方向？
3. 不同角色的内核关键词是否有明显区分，而非都是"温柔善良"之类的通用词？
4. 视觉禁忌是否有效排除了与角色气质不符的外貌方向？

【粘贴角色内核提取内容】

逐条说明问题。
```

##### D. 返修优化版

```
根据自查意见修改角色内核提取。将无法可视化的词汇替换为具体可描绘的特质，区分不同角色的关键词，补充视觉禁忌。

原始提取：【粘贴原始内核提取】
自查意见：【粘贴自查结果】
```

#### 1.5 交付给下游节点

将每个角色的内核关键词组完整复制，交给节点 2（年龄身份校准专家）。节点 2 将基于内核关键词和身份背景，判断年龄和时代对外貌的具体影响。

#### 1.6 人工验收清单

- [ ] 每个角色的性格关键词是否都能转化为视觉元素？
- [ ] 不同角色的气质方向是否有明显区分？
- [ ] 是否为每个角色标注了视觉禁忌（不适合的外貌元素）？

---

### 节点 2：年龄身份校准专家

#### 2.1 节点定位

确认角色的年龄、时代背景和社会身份对外貌设计的具体影响，为外貌特征设计设置精确的约束条件。同样是"20岁女性"，古代贵族和现代程序员的外貌气质完全不同——这个节点负责将这些差异显式化。

#### 2.2 输入与输出

**输入**：节点 1 的角色内核关键词（含身份标签）

**输出**：每个角色的校准参数（年龄范围 + 时代背景 + 身份对外貌的5项影响说明）

#### 2.3 使用顺序

1. 先用「快速生成版」得到校准参数初稿。
2. 历史背景复杂或需要精确服饰考据时，改用「专家增强版」。
3. 用「自查审稿版」检查校准参数是否与故事设定一致。
4. 有问题则用「返修优化版」修正。
5. 对照 2.6 验收清单确认，通过后交给节点 3。

#### 2.4 提示词包

##### A. 快速生成版

```
你是一位角色设定师。请根据以下角色内核，判断年龄、时代背景对外貌设计的影响。

角色内核：【粘贴节点1的内核关键词】

对每个角色输出：
- 推荐年龄范围（精确到5年区间，例如：22-27岁）
- 时代背景说明（例如：现代都市 / 民国上海 / 架空古代）
- 身份对外貌的3项影响（例如：贵族身份→皮肤白皙、手部保养好、无劳动痕迹）
```

##### B. 专家增强版

```
你是一位具有服装史和造型设计背景的角色设定专家。

任务：基于角色内核，为每个角色提供精确的年龄与身份校准参数，确保外貌设计符合时代和身份逻辑。

输入：
- 角色内核关键词：【粘贴节点1的完整输出】

校准维度：
1. 精确年龄（精确到年龄区间，并说明判断依据）
2. 时代背景（说明对外貌的具体影响，不只是标注"古代"）
3. 社会阶层对外貌的影响（肤色、手部、体态、面部状态）
4. 职业对身体特征的影响（武将有肌肉、文人有书卷气等）
5. 生活状态对外貌的影响（长期逃亡者→消瘦疲惫；贵族养尊处优→丰腴气色好）

约束：所有影响说明必须是可以画出来的外貌元素，不能是心理描述。

输出：每个角色的校准参数表（5个维度）
```

##### C. 自查审稿版

> 此为当前节点的自查模式，不是新的专家角色。

```
请检查以下年龄身份校准是否存在以下问题：
1. 年龄范围是否精确（不能只写"青年"，需要具体区间）？
2. 身份对外貌的影响是否都是可绘制的视觉元素（不能是心理描述）？
3. 校准参数与故事文本中对角色的描述是否存在矛盾？
4. 不同角色因身份差异导致的外貌差异是否被充分挖掘？

【粘贴年龄身份校准内容】

逐条说明问题。
```

##### D. 返修优化版

```
根据自查意见修正校准参数。将心理描述改为可绘制的视觉元素，修正与故事设定矛盾的参数，增加身份差异对外貌的影响说明。

原始校准：【粘贴原始校准参数】
自查意见：【粘贴自查结果】
```

#### 2.5 交付给下游节点

将每个角色的校准参数完整复制，交给节点 3（外貌特征设计专家）。节点 3 将在校准参数的约束范围内设计具体外貌特征。

#### 2.6 人工验收清单

- [ ] 每个角色的年龄是否精确到 5 年区间？
- [ ] 身份和职业对外貌的影响是否都是可以画出来的视觉元素？
- [ ] 校准参数是否与故事文本中的角色描述一致，没有矛盾？

### 节点 3：外貌特征设计专家

#### 3.1 节点定位

在角色内核和校准参数的基础上，设计每个角色的详细外貌特征，包括面部五官、体型身材、肤色发色等。这个节点的输出是后续提示词生成的直接素材，描述必须精确到 AI 绘图工具可以理解的程度。

#### 3.2 输入与输出

**输入**：节点 1 角色内核关键词 + 节点 2 校准参数

**输出**：每个角色的详细外貌描述（面部五官 + 体型 + 肤色 + 发型发色 + 气质表现）

#### 3.3 使用顺序

1. 先用「快速生成版」得到外貌描述初稿。
2. 需要更精细的五官设计或特定艺术风格参照时，改用「专家增强版」。
3. 用「自查审稿版」检查描述是否足够精确、与角色内核是否一致。
4. 有问题则用「返修优化版」修正。
5. 对照 3.6 验收清单确认，通过后交给节点 4。

#### 3.4 提示词包

##### A. 快速生成版

```
你是一位角色外貌设计师。请根据以下角色内核和校准参数，设计每个角色的详细外貌。

角色内核：【粘贴节点1输出】
校准参数：【粘贴节点2输出】
视觉风格：【填入：写实/动漫/古风/赛博朋克等】

对每个角色输出：
- 面部：脸型、眼睛（形状+颜色）、鼻子、嘴唇、眉毛
- 体型：身高、体型（纤细/健壮/中等）
- 肤色：具体色调（例如：象牙白、小麦色、古铜色）
- 发型：长度、造型、发色
- 气质特点：常态表情倾向和体态特征
```

##### B. 专家增强版

```
你是一位专业角色设定师，同时熟悉AI绘图工具的视觉描述逻辑。

任务：基于角色内核和校准参数，设计每个角色的精确外貌特征，所有描述必须能直接转化为AI绘图提示词。

输入：
- 角色内核关键词：【粘贴节点1输出】
- 年龄身份校准参数：【粘贴节点2输出】
- 目标视觉风格：【填入风格】
- 参考风格描述（可选）：【填入参考图描述】

设计要求：
- 面部特征：精确到五官各部分的具体形态（不能只写"好看的眼睛"）
- 体型描述：使用可量化的词汇（170cm纤细体型，而非"高挑美丽"）
- 肤色：使用颜色名称而非程度词（象牙白，而非"很白"）
- 发型：说明长度、卷直度、发际线特征、扎发方式（如有）
- 标志性特征：1-2个让这个角色在人群中一眼可认的特殊外貌元素

约束：所有描述必须是AI绘图工具能理解的视觉元素，禁止使用情感形容词代替外貌描述。

输出：每个角色的精确外貌设定表（6个维度）+ 标志性特征说明
```

##### C. 自查审稿版

> 此为当前节点的自查模式，不是新的专家角色。

```
请检查以下外貌特征设计是否存在以下问题：
1. 是否有用情感词代替外貌描述的情况（例如"温柔的眼神"而不是"眼尾微垂，单眼皮"）？
2. 外貌特征是否与角色内核和性格气质一致（冷酷角色不该有甜美大眼）？
3. 每个角色是否有至少1个标志性的、一眼可认的外貌特征？
4. 外貌描述是否足够精确，可以直接转化为AI绘图提示词词汇？

【粘贴外貌特征设计内容】

逐条说明问题。
```

##### D. 返修优化版

```
根据自查意见修改外貌设计。将情感词替换为具体外貌元素，修正与角色内核不一致的外貌特征，为缺少标志性特征的角色补充。

原始设计：【粘贴原始外貌描述】
自查意见：【粘贴自查结果】
```

#### 3.5 交付给下游节点

将每个角色的完整外貌设定表复制，交给节点 4（服装道具设计专家）。节点 4 将在外貌基础上设计与角色气质匹配的服装和道具。

#### 3.6 人工验收清单

- [ ] 所有外貌描述是否都是可绘制的视觉元素，没有情感词替代？
- [ ] 每个角色是否有至少 1 个一眼可认的标志性外貌特征？
- [ ] 外貌特征是否与角色内核（性格/动机/气质）保持一致？

### 节点 4：服装道具设计专家

#### 4.1 节点定位

在外貌特征基础上，为每个角色设计与其身份、时代和性格匹配的服装方案和标志性道具。服装是角色最直观的身份符号，也是多场景下角色识别的重要线索。

#### 4.2 输入与输出

**输入**：节点 2 校准参数（时代/身份）+ 节点 3 外貌特征设计

**输出**：每个角色的服装设计方案（主服装 + 配色方案 + 标志性道具 + 不同场景换装建议）

#### 4.3 使用顺序

1. 先用「快速生成版」得到服装设计初稿。
2. 有历史服饰要求或精细风格要求时，改用「专家增强版」。
3. 用「自查审稿版」检查服装是否与身份时代匹配。
4. 有问题则用「返修优化版」修正。
5. 对照 4.6 验收清单确认，通过后交给节点 5。

#### 4.4 提示词包

##### A. 快速生成版

```
你是一位服装设计师。请根据以下角色信息，设计每个角色的服装方案。

校准参数（时代/身份）：【粘贴节点2输出】
外貌设定：【粘贴节点3输出】
视觉风格：【填入风格】

对每个角色输出：
- 主服装描述（款式、材质、颜色）
- 配色方案（主色 + 辅色，例如：深蓝主色+金色配饰）
- 标志性道具（1-2件能体现身份或性格的物品）
- 日常/正式场合的换装简述
```

##### B. 专家增强版

```
你是一位兼具服装史知识和视觉设计能力的角色服装设计师。

任务：为以下角色设计完整的服装方案，确保服装在视觉上强化角色的身份和气质，同时满足AI绘图的描述精度要求。

输入：
- 校准参数：【粘贴节点2的时代/身份校准】
- 外貌特征：【粘贴节点3的完整外貌设定】
- 视觉风格：【填入风格】

设计要求：
- 款式细节：精确到衣领、袖型、腰部处理、下摆长度等具体元素
- 材质质感：说明材质视觉感（飘逸薄纱/厚重丝绒/粗糙麻布等）
```

## 常见错误

**错误 1：跳过内核提取直接设计外貌**

表现：生成的角色外貌看起来好看，但与故事中的人物气质完全不符，导致读者/观众认为"这不是我想象中的那个角色"。

修复：必须先完成节点 1，从故事文本中提取角色内核，再以内核为约束设计外貌。

**错误 2：多角色外貌设计雷同**

表现：主角和女配角都是"黑色长发、白皙皮肤、纤细身材"，AI 生成时不断把两个人画成同一个人。

修复：必须执行节点 7 的差异化审查，为每对相似角色在至少 2 个维度上强制拉开差距。

**错误 3：三视图提示词不一致**

表现：正面图生成出来是蓝色眼睛，侧面图变成了棕色眼睛，背面图又改了发型。

修复：执行节点 5 的自查审稿版，确认三张图的固定外貌词完全相同，然后执行节点 6 建立词库确保后续生成的一致性。

## 延伸玩法

- **变体 1：多服装版**：用节点 4 的换装建议，为同一角色生成不同场景服装版本的提示词（战斗服/正装/便服）。
- **变体 2：表情包版**：在三视图基础上，用节点 8 的场景模板生成同一角色的多种表情变体（生气/开心/悲伤/惊讶）。
- **进阶组合**：与"工作流 04（短剧策划工作流）"结合，用本工作流的角色档案直接对接短剧人物关系节点的输出，形成"策划→角色设定→开始绘制"的完整链路；与"工作流 06（视频分镜工作流）"结合，将节点 6 的一致性词库直接用于分镜中的角色连续性控制。

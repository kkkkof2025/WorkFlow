# 专家节点提示词生成

## 用途

用于为单个专家节点生成全套提示词（基础/进阶/审查/修正，以及可选的极简版和专家模式版）。

---

## 标准提示词（生成 4 类）

```
你是一位 Prompt 工程师。请为以下专家节点生成完整的 4 类提示词。

**节点名称**：【填入节点名称，例如：故事扩写专家】
**所在工作流**：【填入工作流名称】
**节点职责**：【填入职责说明】
**节点输入**：【填入输入内容】
**节点输出**：【填入输出内容】

请生成以下 4 类提示词：

**L1 基础提示词**（50～100 字）
目标用户：普通职场人，直接复制使用。
要求：角色简单明确，任务清晰，输出格式具体。

**L2 进阶提示词**（100～200 字）
目标用户：有经验的 AI 用户，追求更高质量。
要求：提供更多上下文约束，明确风格、结构、质量标准。

**L3 审查提示词**（50～150 字）
用途：检查 L1 或 L2 的输出是否存在质量问题。
要求：列出 3～5 个具体审查维度，每条审查都是可判断的问题。

**L4 修正提示词**（50～100 字）
用途：基于 L3 审查结果，要求 AI 重写输出。
要求：明确"保留什么，改变什么"，避免推倒重来。

每类提示词单独输出，格式为代码块。
```

## 进阶版（生成 6 类，含极简和专家模式）

```
你是一位 Prompt 工程师。请为以下专家节点生成 6 类提示词。

**节点名称**：【填入节点名称】
**节点职责**：【填入职责说明】
**节点输入**：【填入输入内容】
**节点输出**：【填入输出内容】

请生成以下 6 类提示词：

1. **极简提示词**（20 字以内）：用于快速场景，质量要求不高时使用。
2. **基础提示词**（50～100 字）：普通用户直接使用。
3. **进阶提示词**（100～200 字）：复杂场景，专业用户。
4. **审查提示词**（50～150 字）：检查输出质量。
5. **修正提示词**（50～100 字）：基于审查结果重写。
6. **专家模式提示词**（200 字以上）：针对高要求场景，要求 AI 以资深专家身份，结合行业知识深度处理任务。

每类提示词单独输出代码块，并说明适用场景。
```

## 批量生成版（一次生成多个节点的提示词）

```
你是一位 Prompt 工程师。请为以下工作流中的所有节点，分别生成 4 类提示词（基础/进阶/审查/修正）。

**工作流名称**：【填入工作流名称】
**节点列表**：
1. 【节点 1 名称】- 职责：【职责】- 输入：【输入】- 输出：【输出】
2. 【节点 2 名称】- 职责：【职责】- 输入：【输入】- 输出：【输出】
...

按节点顺序逐一输出，每个节点输出 4 个代码块。
不要省略任何节点，不要合并节点。
```

# 质量检查提示词

## 用途

用于检查工作流卡是否真实、具体、可执行、可验收，并给出评分。

---

## 提示词

```
你是一位内容质量评审专家。请对以下工作流卡进行质量评分，满分 100 分。

【粘贴工作流卡内容】

评分维度（每项 20 分）：

1. **场景真实性**（20 分）
   - 场景是否基于真实工作情境，而非假设场景？
   - 描述是否具体到可以对号入座？
   - 扣分点：场景模糊、假设性过强、无法判断适用人群

2. **节点完整性**（20 分）
   - 每个节点是否都有：职责、输入、输出、4 类提示词、验收标准？
   - 有无节点被草率处理或直接跳过？
   - 扣分点：节点信息不完整、提示词少于 4 类

3. **提示词可用性**（20 分）
   - 每条提示词是否可以直接复制并得到有效输出？
   - 是否存在需要用户自行理解才能使用的占位符或模糊表达？
   - 扣分点：提示词含占位符、模糊指令、无法直接运行

4. **交接清晰度**（20 分）
   - 节点间的交接是否明确说明"上游输出什么，下游接收什么"？
   - 是否存在信息断层或交接遗漏？
   - 扣分点：交接说明缺失、交接内容模糊

5. **验收可操作性**（20 分）
   - 验收清单是否能被非专业人员理解和执行？
   - 每条是否都是可判断的是/否问题？
   - 扣分点：验收条件模糊、需要专业判断才能完成

输出格式：
- 各维度得分 + 扣分理由
- 总分
- 是否达到合格线（80 分）
- 优先修改建议（3 条，按影响力排序）
```

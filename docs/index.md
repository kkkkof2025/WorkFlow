# AI 工作流图鉴

欢迎来到 **AI 工作流图鉴**——30 个真实岗位的多专家 AI 协作方法。

!!! tip "这不是提示词大全"
    本书的核心不是提示词，而是**工作流**。每一张工作流卡，都是一套把真实工作拆成多个 AI 专家节点、可执行、可交接、可验收的完整流程。

## 如何使用本书

1. **找到你的岗位**：在左侧导航中按岗位类型找到对应章节。
2. **读懂工作流卡**：每张卡包含场景、目标、专家节点、提示词和验收清单。
3. **按节点逐步执行**：不要跳过节点，每个节点的输出是下一个节点的输入。
4. **用审查和修正提示词做闭环**：每个节点都有审查和修正提示词，用好它们。
